This commercial software provides design tools and algorithms for in silico cloning, codon optimization, back translation and primer design. It supports GenBank and FASTA formats. The same company has also developed another software called ATUM's GPS technology that is a set of all four design, build, test, learn tools and its platform is based on Design of Experiment (DoE), empirical testing and machine learning enabling the biological systems to be optimized for specific properties that matters.
